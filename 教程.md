# 机器人启动器 - Minecraft 机器人控制工具

## 📖 项目简介

这是一个功能全面的Minecraft机器人图形化启动器，支持完全实时控制、多种功能模块和友好的用户界面，适合Minecraft服务器管理和自动化操作。

## ✨ 功能特点

### 🎮 完全实时控制
- **按键按下持续控制，释放按键停止控制**
- 100毫秒命令发送间隔，响应即时
- 支持键盘WASD和鼠标点击双重控制

### 🚶 移动控制
- ✅ WASD前后左右移动
- ✅ 跳跃功能（空格键）

### ⚔️ 动作控制
- ✅ 左键攻击（仅攻击实体，不会挖方块）
- ✅ 右键使用/交互
- ✅ Q丢弃物品
- ✅ F切换主副手

### 🎒 物品栏控制
- ✅ 1-9数字键直接切换物品栏

### 🤖 模式控制
- ✅ 守卫模式
- ✅ 跟随模式
- ✅ 战斗模式
- ✅ 停止所有动作

### 💬 聊天与状态
- ✅ 发送聊天消息
- ✅ 查看机器人情绪
- ✅ 聊家常功能
- ✅ 查看帮助信息

### 📝 记忆功能
- ✅ 支持15种记忆类型查询
- ✅ 下拉选择记忆类型

### 📊 投影功能
- ✅ 创建投影，支持自定义文件名

### 🔫 PVP功能
- ✅ 精打击指定玩家
- ✅ 输入目标玩家名称

### 📍 位置查询
- ✅ 查询机器人当前位置
- ✅ 查询指定玩家位置

## 📋 系统要求

### 硬件要求
- CPU: 双核以上
- 内存: 2GB以上
- 存储空间: 500MB以上

### 软件要求
- **Node.js 16+**: 运行机器人核心代码
- **Python 3.6+**: 运行图形化启动器

## 🚀 安装步骤

### 1. 安装必要软件

#### Node.js安装
1. 访问 [Node.js 官方网站](https://nodejs.org/zh-cn/)
2. 下载并安装「稳定版」
3. 验证安装：
   ```bash
   node --version
   ```

#### Python安装
1. 访问 [Python 官方网站](https://www.python.org/downloads/)
2. 下载并安装「稳定版」
3. **务必勾选「Add Python to PATH」**
4. 验证安装：
   ```bash
   python --version
   ```

### 2. 安装项目依赖

1. 进入项目文件夹：
   ```bash
   cd 项目文件夹路径
   ```

2. 安装Node.js依赖：
   ```bash
   npm install
   ```

3. Python依赖（可选，一般内置）：
   ```bash
   pip install tk
   ```

## 🎯 使用说明

### 1. 启动机器人核心

```bash
node index.js
```

### 2. 启动图形化启动器

```bash
python bot_launcher_simple.py
```

### 3. 配置连接信息

1. 切换到「启动」标签页
2. 输入服务器IP（如 `h.rainplay.cn`）
3. 输入服务器端口（如 `26165`）
4. 输入机器人名称（如 `dream216`）
5. 输入登录密码
6. 点击「保存设置」

### 4. 启动机器人

- 点击「启动机器人」按钮
- 看到状态栏显示「机器人运行中...」，说明连接成功

### 5. 控制机器人

- 切换到「控制」标签页
- 使用键盘WASD和鼠标控制机器人
- 向下滚动查看所有功能

## ❓ 常见问题

### 1. 启动器无法打开
- 检查Python是否正确安装
- 确保勾选了「Add Python to PATH」
- 尝试安装tkinter：`pip install tk`

### 2. 机器人无法连接到服务器
- 检查IP和端口是否正确
- 检查服务器是否在线
- 检查防火墙设置

### 3. 控制命令没有反应
- 确保机器人核心正在运行
- 检查机器人是否成功连接到服务器
- 检查启动器和机器人核心是否在同一台电脑

### 4. 界面无法滚动
- 调整窗口大小
- 使用鼠标滚轮滚动
- 使用右侧滚动条
